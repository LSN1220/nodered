# 样例项目目录说明

这个目录用于存放后续生成的独立可运行样例项目。

设计原则：

- 每个目录是一个独立样例
- 每个样例只讲一个核心主题
- 每个样例都应该可以直接运行
- 每个样例都应包含详细注释
- 每个样例都应自带说明文档

建议顺序：

1. `01-js-basics` 已创建
2. `02-nodejs-async` 已创建
3. `03-express-routing` 已创建
4. `04-express-middleware`
5. `05-express-crud-with-db`
6. `06-db-schema-admin`
7. `07-db-schema-iot`
8. `08-nodered-http-flow`
9. `09-nodered-mqtt-flow`
10. `10-admin-api-demo`
11. `11-iot-flow-demo`

后续如果继续推进，建议从 `01-js-basics` 开始生成第一批样例。
